# Dynamic Exam Scheduling System

This project is a desktop-based application designed to automatically generate
exam schedules by considering courses, students, instructors, classrooms, and
capacity constraints.

## Features
- Role-based access (Admin & Department Coordinator)
- Excel-based import for courses and student data
- Automatic exam scheduling with conflict prevention
- Classroom capacity and availability checks
- Visualized exam schedules and seating plans
- Export schedules and seating plans as PDF or Excel

## Technologies
- Programming Language: Python
- Data Handling: Excel parsing
- Focus Areas: Algorithms, Optimization, Constraint-Based Scheduling

## Purpose
This project was developed as part of the Programming Laboratory course to
strengthen algorithm design, problem-solving, and practical software development skills.
